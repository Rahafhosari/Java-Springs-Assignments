CONNECTING DATABASES
when creating the project add the dependancies
1- Spring Boot DevTools
2-Spring Web
3-Spring Data JPA
4-MySQL Driver 

 (Or add them manually in the pom.xml file 
<dependency>
    <groupId>mysql</groupId>
    <artifactId>mysql-connector-java</artifactId>
    <scope>runtime</scope>
</dependency>
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-data-jpa</artifactId>
</dependency>)


Edit 
src/main/resources/applications.properties >>
spring.datasource.url=jdbc:mysql://localhost:3306/<<YOUR_SCHEMA>> [name of schema]
spring.datasource.username=<<dbuser>> [root]
spring.datasource.password=<<dbpassword>> [root]
spring.datasource.driver-class-name=com.mysql.jdbc.Driver
spring.jpa.hibernate.ddl-auto=update

if there was errors:
1- append time zone at (spring.datasource.url):
 example : spring.datasource.url=jdbc:mysql://localhost:3306/<<Your-Schema-Name>>?serverTimezone=UTC

2- error in deprecation warning
 remove (spring.datasource.driver-class-name=com.mysql.jdbc.Driver) from settings of application.properties
